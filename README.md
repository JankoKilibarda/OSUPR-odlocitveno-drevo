# OSUPR-odlocitveno-drevo

## Opis projekta

Projekt vsebuje lastno implementacijo algoritma odločitvenega drevesa
za binarno klasifikacijo. Implementirana sta kriterija Gini in entropija.

Algoritem je preizkušen na podatkovnih zbirkah Titanic in
Breast Cancer Wisconsin. Rezultati lastne implementacije so primerjani
z implementacijo DecisionTreeClassifier iz knjižnice scikit-learn.

## Potrebne knjižnice

- numpy
- pandas
- scikit-learn

Namestitev knjižnic:

pip install -r requirements.txt

## Zagon programa

python "JankoKilibarda projekat OSUPR.py"
